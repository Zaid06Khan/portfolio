# Zaid Khan - Portfolio

Live site: [zkhan.vercel.app](https://zkhan.vercel.app/)

Personal portfolio built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

npm install && npm run dev

Open https://zkhan.vercel.app/ to see the live site.

## Tech Stack

- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Deployment: Vercel
